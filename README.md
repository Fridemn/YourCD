# YourCD

YourCD 是一个面向 Android 的离线 Flutter 小应用，用“技能冷却”的方式记录生活中的自律事项。

## 功能

- 用户自建技能：名称、备注、颜色、预置图标或本地图片。
- 三类 CD 规则：使用后冷却指定时长、每天固定时间重置、每周固定时间重置。
- 点击使用后显示倒计时和可用状态。
- 冷却中仍可强制使用，并记录强制使用时间。
- 本地保存技能和使用记录，不需要联网。
- 可开启冷却结束本地通知。

## 运行

```bash
flutter run
```

## 检查

```bash
flutter analyze
flutter test
```
